## Searching

**Use `fts-search`** when looking for keywords across text content (title, description,
business_context, story). This searches the full text of those fields:
```bash
python3 scripts/db_cli.py fts-search '{"query":"billing sprint","limit":10}'
# Supports: phrases "open rate", OR, prefix match statistic*
```

**Use `list-{{MAIN_TABLE}}`** when filtering by structured fields — status, tag, or date:
```bash
python3 scripts/db_cli.py list-{{MAIN_TABLE}} '{"status":"todo"}'
python3 scripts/db_cli.py list-{{MAIN_TABLE}} '{"tag":"sprint-147"}'
python3 scripts/db_cli.py list-{{MAIN_TABLE}} '{"since":"2026-01-01","limit":20}'
```

**Use `query`** for ad-hoc SELECTs not covered by named commands:
```bash
python3 scripts/db_cli.py query '{"sql":"SELECT * FROM {{MAIN_TABLE}} WHERE status = ?","params":["todo"]}'
```
